## Loader
Displays the current state of some asynchronous operation.

### Children: 
 * `message` - Indicates the reason for this asynchronous state.
 * `progress` - Visual indication that the operation is ongoing. E.g. spinner, percentage, bar.

### Core Modifiers: 
 * `spinner`
 * `progress`
